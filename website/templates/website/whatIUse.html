Коротко об инструментах разработки.

Итак, пройдёмся по списку:

  • IDE: Visual Studio 2013 - вне конкуренции для разработки под .NET. Работать с Node JS и JavaScript тоже вполне приятно.
  • VCS: MS TFS, Mercurial для работы, Git - для дома. Разработка под .NET автоматически ставит на путь использования технологий Microsoft.
  • DB: MS SQL Server 2014, а для разработки SQL Server Management Studio 2014.
  • Delphi IDE: Rad Studio 10 - в ногу со временем.
  • Autobuild: MS TFS Build service - отличная работа "из коробки" с TFS с привязкой к changeset и work item.
  • Release: MS Release Management - без него был бы не полный набор инструментов от Microsoft.
  • Bug tracker: MS TFS, Atlassian Jira - в зависимости от проекта.
  • Monitoring: Zabbix - System Center Operations Manager уже на подходе, но пока Zabbix.
  • Logs: Elasticsearch + kibana - красиво, удобно, гибко.
  • Docs: Atlassian Confluence - иногда приходится писать документацию к проектам.
  • Tools: SoapUI для разработки Web API, Wireshark для тяжелых случаев.
  • Other: Всё остальное в Nodepad++.

В общем вполне логичный набор для C# разработчика.
