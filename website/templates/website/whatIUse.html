Коротко об инструментах разработки.

Итак, пройдёмся по списку:

  • IDE: Visual Studio 2017 - вне конкуренции для разработки под .NET. Работать с Node JS и JavaScript тоже вполне приятно.
  • Front-end IDE: WebStorm 2016 - самый комфортный инструмент для связки NodeJS + CSS + HTML + JS.
  • Python IDE: PyCharm 2017 - логичный выбор для пользователей WebStorm.
  • Delphi IDE: Rad Studio 10 - в ногу со временем.
  • DB: MS SQL Server 2016, а для разработки SQL Server Management Studio 2016.
  • Autobuild: MS TFS Build service - отличная работа "из коробки" с TFS с привязкой к changeset и work item. Для дома - Travis CI
  • Deployment: MS Release Management - без него был бы не полный набор инструментов от Microsoft.
  • VCS: MS TFS, Mercurial, Git - в основном конечно же Git, но есть и старые репозитории.
  • Containers: Docker - под Linux только Docker.
  • Bug tracker: MS TFS, Atlassian Jira - в зависимости от проекта.
  • Monitoring: Zabbix - System Center Operations Manager уже на подходе, но пока Zabbix.
  • Logs: Elasticsearch + kibana - красиво, удобно, гибко.
  • Docs: Atlassian Confluence - иногда приходится писать документацию к проектам.
  • Tools: SoapUI для разработки Web API, Wireshark для тяжелых случаев.
  • Other: Всё остальное в Nodepad++.

В общем вполне логичный набор для разработчика.
